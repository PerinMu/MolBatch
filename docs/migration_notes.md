# Migration Notes

This repository was generated by refactoring a single-user PyMOL batch script into a modular package.
The original script focused on:

- Loading many CIF files from one folder
- Selecting a reference structure by file stem
- Aligning all structures by chain A
- Keeping reference core chains and display chains
- Showing compact ligand-style chains with object-level coloring

Those behaviors are preserved in the `comparison` preset and the PyMOL backend.
