# CLI reference

## Commands

- `molbatch init-config`
- `molbatch validate`
- `molbatch scan`
- `molbatch plan`
- `molbatch generate`
- `molbatch run`
- `molbatch list-presets`
- `molbatch list-backends`
- `molbatch doctor`

## Typical workflow

1. Create a starter config.
2. Adjust paths, chains, and style options.
3. Scan the input directory.
4. Inspect the normalized plan.
5. Generate scripts for PyMOL or ChimeraX.
6. Optionally execute the script with a local viewer installation.
