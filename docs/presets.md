# Presets

## comparison
Default comparison mode for side-by-side inspection.

## ligand-focus
Emphasizes displayed chains with slightly thicker sticks.

## presentation
Adds labels and enables both session and image export.

## minimal
Disables extra outputs for the smallest possible run.

## publication
Optimized defaults for static figure export.

## screening
Optimized defaults for large recursive directory scans and report capture.
